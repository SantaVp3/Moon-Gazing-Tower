# Moon Gazing Tower Documentation

This directory contains the documentation for the Moon Gazing Tower project, built with [VitePress](https://vitepress.dev/).

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run docs:dev
   ```

3. Build for production:
   ```bash
   npm run docs:build
   ```

4. Preview the production build:
   ```bash
   npm run docs:preview
   ```

## Structure

- `index.md`: The home page.
- `.vitepress/config.mts`: Configuration file.
- `guide/`: Documentation guides.
- `api/`: API reference.
