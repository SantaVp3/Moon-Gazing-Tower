# 介绍

Moon-Gazing-Tower (望月塔) 是一款现代化的自动化安全扫描平台，集成资产管理、漏洞扫描、任务调度等功能。

## 核心特性

### 🛡️ 资产管理
- **多类型支持**：支持域名、IP、URL、服务等多种资产类型。
- **分组管理**：灵活的资产分组，方便管理不同项目的资产。
- **标签系统**：自定义标签，快速筛选和定位资产。
- **批量操作**：支持资产的批量导入和导出。

### ⚡ 任务管理
- **灵活输入**：支持直接输入目标（IP、域名、URL、CIDR）。
- **多种扫描模式**：
  - 子域名枚举
  - 端口扫描
  - 漏洞扫描
  - 指纹识别
- **调度系统**：支持任务调度与状态实时跟踪。
- **实时推送**：扫描结果通过 WebSocket 实时推送到前端。

### 🔍 漏洞管理
- **仪表盘**：直观的漏洞统计仪表盘。
- **分级管理**：严重、高危、中危、低危、信息五个等级。
- **全生命周期**：待处理、已确认、已修复、已忽略、误报等状态管理。
- **漏洞验证**：支持对发现的漏洞进行二次验证。

### 🛠️ 扫描能力集成
平台集成了多款业界优秀的开源安全工具：
- **子域名枚举**：集成 [subfinder](https://github.com/projectdiscovery/subfinder)
- **端口扫描**：集成 [rustscan](https://github.com/RustScan/RustScan)
- **Web 爬虫**：集成 [katana](https://github.com/projectdiscovery/katana)、[rad](https://github.com/chaitin/rad)
- **漏洞扫描**：集成 [Nuclei](https://github.com/projectdiscovery/nuclei)
- **指纹识别**：内置指纹库
- **CDN 检测**：智能识别 CDN 节点
- **子域名接管**：检测子域名接管风险

### 🔌 第三方集成
支持配置多种第三方 API 以增强扫描能力：
- FOFA
- Hunter
- Quake
- SecurityTrails
- Crt.sh

### 🔔 其他功能
- **页面监控**：监控目标页面变化。
- **通知系统**：支持邮件通知。
- **实时推送**：WebSocket 实时消息推送。
- **安全认证**：基于 JWT 的用户认证系统。
