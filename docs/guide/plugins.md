# 插件管理 (Plugin Management)

Moon-Gazing-Tower 提供了强大的插件系统，允许用户扩展扫描能力、集成第三方工具或自定义检测逻辑。

## 插件架构

插件系统基于松耦合设计，支持以下类型的插件：

1. **扫描插件**: 集成新的扫描工具（如 SQLMap, Xray 等）。
2. **指纹插件**: 添加新的指纹识别规则。
3. **POC 插件**: 添加新的漏洞验证脚本（支持 Nuclei 模板）。
4. **辅助插件**: 提供数据处理、格式转换等辅助功能。

## 插件管理功能

### 1. 插件列表
查看当前系统中安装的所有插件，包括：
- **名称**: 插件的唯一标识。
- **版本**: 当前安装的版本。
- **状态**: 已启用/已禁用。
- **类型**: 扫描器、指纹、POC 等。

### 2. 安装插件
支持多种安装方式：
- **上传安装**: 上传插件包（.zip 或 .tar.gz）。
- **在线安装**: 从官方插件市场直接下载安装。
- **本地加载**: 加载服务器本地目录下的插件。

### 3. 配置插件
每个插件可以有独立的配置项：
- **API Key**: 第三方服务的密钥。
- **并发数**: 插件执行时的并发限制。
- **超时时间**: 执行超时设置。
- **自定义参数**: 传递给插件的额外参数。

### 4. 插件开发

开发者可以编写自定义插件来扩展系统能力。

#### 插件结构
```json
{
  "name": "my-custom-scanner",
  "version": "1.0.0",
  "type": "scanner",
  "entry": "main.py",
  "config": {
    "timeout": 10
  }
}
```

#### 接口规范
插件需遵循标准输入输出规范：
- **输入**: JSON 格式的目标信息。
- **输出**: JSON 格式的扫描结果。

## 常用插件

| 插件名称 | 类型 | 描述 |
|----------|------|------|
| **Nuclei** | 扫描器 | 强大的基于模板的漏洞扫描器 |
| **Subfinder** | 扫描器 | 被动子域名枚举工具 |
| **FingerprintHub** | 指纹 | 包含 10000+ Web 指纹库 |
| **Xray-Poc** | POC | 集成 Xray 社区版 POC |
